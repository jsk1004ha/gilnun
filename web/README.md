# 길눈 AI 웹 데모

Android APK와 같은 의미 계약을 브라우저에서 직접 조작할 수 있게 구현한 공개 데모입니다.
실제 개인정보, 계정, 외부 API, 영구 브라우저 저장소를 사용하지 않습니다.

## 실행

Node.js 22.13 이상에서:

```bash
npm install
npm run dev
```

production 빌드와 렌더 검증:

```bash
npm run build
npm test
```

## 시연 흐름

1. 학습자 모드에서 `임시 저장`을 6초 안에 세 번 누릅니다.
2. 도움 확인을 수락하고 도우미가 `신청 내용 확인` 한 지점을 선택합니다.
3. 여섯 의미 필드로만 구성된 `PatchV1`을 저장합니다.
4. 레이아웃 B에서 패치를 재사용하고, 사용자가 강조된 버튼을 직접 누릅니다.
5. `review-ready` 사후조건이 확인된 경우만 `VERIFIED` 영수증이 발급됩니다.

리비전이나 의미 필드가 하나라도 다르면 강조와 실행을 모두 중단합니다.
