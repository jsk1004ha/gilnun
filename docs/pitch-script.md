# 길눈 AI 4분 발표 대본

“길눈은 범용 AI가 아닙니다. 고령 사용자가 막힌 지점을 발견하고, 한 번 검증한 해결을 참여 서비스 안에서 안전하게 재사용하는 온디바이스 안내 계층입니다.”

고령 사용자는 전체 과정을 모르는 것이 아니라 작은 버튼 하나를 놓쳐 특정 단계에서 멈추는 경우가 많습니다. 길눈은 사용자를 평가하지 않습니다. 같은 비진행 버튼을 세 번 누르는 마찰 후보를 발견한 뒤 먼저 도움 의사를 묻습니다.

지금 큰 임시 저장 버튼을 세 번 누릅니다. 진행 상태는 바뀌지 않았고 길눈이 도움 여부를 묻습니다. 아직 패치가 없으므로 같은 APK의 청년 도우미 모드로 바꿉니다. 도우미는 전체 매뉴얼 대신 `신청 내용 확인` 한 지점만 해결합니다.

저장되는 것은 화면 좌표가 아닙니다. 화면 ID, 호환 버전, 안정 키, 역할, 접근성 이름, 예상 결과뿐입니다. 레이아웃 B에서는 버튼 위치와 글자 크기, 카드 순서가 바뀌었습니다. 비교용 좌표 마커는 빗나가지만 길눈은 여섯 의미 필드가 모두 맞는 단 하나의 버튼만 강조합니다.

앱은 대신 누르지 않습니다. 사용자가 직접 누르고 실제로 `review-ready` 상태가 확인된 뒤에만 완료 영수증을 냅니다. 영수증은 안내를 보여준 것, 사용자의 행동을 관찰한 것, 결과를 검증한 것을 따로 표시합니다.

검증된 성공 뒤에는 도움을 한 단계만 줄입니다. 다시 어려우면 즉시 올립니다. 이는 장기 숙련을 입증하는 모델이 아니라 시연 가능한 가역 규칙입니다.

이 MVP에는 다른 앱 접근, 클라우드 AI, 실제 개인정보, 결제·동의·최종 제출 자동화가 없습니다. 비행기 모드에서 동작하는 통제된 연구 시제품입니다.

“길을 대신 걸어주는 AI가 아니라, 모두가 다시 막히지 않도록 디지털 길을 고치는 AI입니다.”
