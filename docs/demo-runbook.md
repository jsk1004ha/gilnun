# 길눈 AI 4분 데모 런북

## 준비

1. `artifacts/gilnun-mvp.apk`를 새로 설치하고 비행기 모드를 켠다.
2. 앱을 열어 `Demo Reset`을 실행한다.
3. 학습자·레이아웃 A·도움 3단계·패치 없음인지 확인한다.

## 시연 순서

1. 좌표 비교 표시를 켜고 A의 좌표가 B에서 빗나감을 “비교용 시뮬레이션”이라고 설명한다.
2. A 학습자에서 큰 `임시 저장`을 6초 안에 세 번 누른다.
3. “도움이 필요하신가요?”에서 도움 받기를 선택한다.
4. 패치 없음 안내 뒤 같은 앱의 도우미 모드로 전환한다.
5. 도우미가 `신청 내용 확인` 한 지점을 선택하고 PatchV1의 여섯 의미 필드를 확인해 저장한다.
6. 학습자·레이아웃 B로 전환해 이동된 `review-next`만 강조되는 것을 보인다.
7. 사용자가 직접 버튼을 누른 뒤 안내 표시·사용자 행동 관찰·사후조건 검증이 분리된 VERIFIED 영수증을 보인다.
8. 도움 단계가 한 단계 낮아진 것을 보이고, 직접 도움 요청으로 즉시 한 단계 복구되는 것을 보인다.

## 실패 안전

invalid fixture를 켜면 대상 강조 없이 “안내를 안전하게 불러오지 못했습니다”가 보여야 한다. 이 상태에서는 완료 영수증이 나오지 않는다.

## 종료

`Demo Reset`으로 5초 안에 초기 상태로 돌아간다. 이 MVP는 통제된 WebView와 온디바이스 규칙 기반 연구 시제품이며 임의 앱 지원이나 장기 학습 효과를 주장하지 않는다.
